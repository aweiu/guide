语句

Regular 除了插值和表达式之外的语法元素就只有语句

Syntax

{#NAME ...}Block..{/NAME}

或者自闭合语句

{#NAME /}

目前 rgl 中只有三类语句: list, if/else/elseif 和 include，未来也会通过相同的语法来进行扩展

1. 😀 迭代的结构不受单节点限制

与大部分其它框架不同的是，包裹在rgl语句包裹结构不限制一个唯一节点内 ，如下例的list语句

new Regular({
  template: `
  {#list items as item}
    {item_index}-<a>{item.name}</a><br/>
  {/list}
  `,
  data: {
    items: [{
        name: 'Apple'
      },
      {
        name: 'Android'
      },
      {
        name: 'Windows'
      }
    ]
  }
}).$inject(document.body)

2. ⚠️ Regular中 语句和XML标签是不能被相互打断的

😔错误️

<div>
{#if true}
  <p>True</p>
</div>
{#else}
  <p>False</p>
</div>
{/if}

😀正确


<div>
{#if true}
  <p>True</p>
{#else}
  <p>False</p>
{/if}
</div>
